# Age Calculator

A responsive Age Calculator built using HTML, CSS, and JavaScript. The application validates user input and calculates the user's age based on the entered birth date.

## Live Demo

https://ibrahim-almahdy.github.io/age-calculator/

## Features

- Responsive Design
- Input Validation
- Age Calculation using JavaScript Date API
- Clean and Simple UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## What I Practiced

- DOM Manipulation
- Form Validation
- JavaScript Date Object
- Event Handling
- Problem Solving
